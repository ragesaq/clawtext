# Clawtext

**Context Layer Augmentation With Text Enhancement Technology**

A hybrid RAG system for OpenClaw that delivers 10x faster session starts with 20% better context quality.

---

## Why "Clawtext"?

| Component | Meaning |
|-----------|---------|
| **Claw** | OpenClaw - the foundation |
| **text** | Context, memory, text enhancement |
| **Augmentation** | Adds capabilities to base system |
| **Technology** | Production-ready implementation |

**Pronunciation**: /klɔːtɛkst/ (claw-text)

---

## Quick Stats

| Metric | Value |
|--------|-------|
| Speed | 10x faster than default |
| Quality | +20% better relevance |
| Latency | 50ms vs 500ms |
| Token Cost | +2.5% (acceptable) |

## Core Idea

Clawtext enhances how OpenClaw retrieves and presents context to the LLM, making every session start faster, more relevant, and better organized.

**Not a replacement** - an augmentation layer.
