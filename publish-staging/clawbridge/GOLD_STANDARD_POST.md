# ClawBridge Gold Standard Post Format

Use this format for any official ClawBridge thread/forum post.

## Structure (required)

1. **Status line**
   - what was extracted
   - source thread id
   - generated artifacts

2. **Short Handoff (inline)**
   - context
   - established
   - open now
   - next

3. **Full Continuity Packet (linked or inline)**
   - objective
   - decisions
   - open questions
   - lane/product context
   - immediate next steps
   - do-not-re-litigate list

4. **Next-Agent Bootstrap (linked or inline)**
   - read-first list
   - first action

5. **Operator note**
   - whether packet is gist-posted
   - whether memory ingest was run

## Quality bar (required)

A post is Gold Standard only if a new agent can answer immediately:
- What are we doing?
- What is already decided?
- What is still open?
- What should I do first?
- What should I avoid re-deriving?

## Safety pass before posting externally

- remove secrets/tokens/internal-only endpoints
- avoid private personal/project details unless explicitly intended
- use secret gist by default
