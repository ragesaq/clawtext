# README Mode Template

Use this when generating a full GitHub README from a seed.

## Instructions
1. Start with your seed artifact (e.g., `seeds/[product]-seed.md`)
2. Fill in all `[PLACEHOLDER]` fields
3. Expand the "How It Works" section with 2-3 feature deep dives
4. Add concrete code examples (copy-paste ready)
5. Keep the narrative spine: Problem → Solution → Deep Dive → Examples → Impact

## Target Length
2-4 KB

## Formatting
- GitHub-flavored Markdown
- Code blocks with language tags
- Tables for feature comparisons
- No Discord-specific formatting

## Review Checklist
- [ ] Does the problem framing make the reader nod along?
- [ ] Is the value prop clear in the first paragraph?
- [ ] Are examples copy-paste ready?
- [ ] Does it end on impact, not features?
- [ ] Would someone understand what this is and why they need it?
