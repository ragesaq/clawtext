# Discord Mode Template

Use this for Discord channel announcements, community updates, or shorter posts.

## Instructions
1. Start with your seed artifact
2. Condense to: Hook → Problem → Solution → Quick Start
3. Use bullet lists (Discord doesn't render tables well)
4. Keep it conversational but professional
5. One key example, link to full docs

## Target Length
0.5-1 KB

## Formatting
- No markdown tables (use bullets instead)
- No headers (# ##) — use **bold** for emphasis
- Emoji sparingly (1-2 max, for visual breaks)
- Wrap links in <> to suppress embeds: `<https://example.com>`

## Review Checklist
- [ ] Does it read naturally in a chat context?
- [ ] Is the value prop clear in the first 2 lines?
- [ ] Are there too many walls of text? (Break them up)
- [ ] Is there a clear next step (install, read more)?

## Example Structure
```
**[Product Name]** — [One-line value prop]

The problem: [2-3 lines describing pain]

What it does: [2-3 lines describing solution]

Quick start:
- `npm install [product]`
- [One key command/example]

More info: <link to full README>
```
