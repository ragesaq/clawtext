## What ClawBridge is for

ClawBridge is for continuity + transfer of active work.

Use it when:
- a thread is too long
- work moves lanes/products
- a new session/agent needs full startup context
- you want structured handoff artifacts instead of ad hoc recaps

## What it is not

ClawBridge is not long-term memory itself.

Use ClawText memory/ingest when the artifact should be durable.
Use ClawBridge when the artifact is for continuity and momentum.
