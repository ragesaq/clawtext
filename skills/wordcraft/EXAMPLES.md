# Wordcraft Examples

## Example 1: Discord Message Refinement

**Original:**
```
hey can you check out this thing i made its pretty cool i think youll like it
```

**Optimized (casual but clearer):**
```
Hey — made something I think you'll like. Want to check it out?
```

**Optimized (more specific):**
```
Hey, I just finished [X]. Think you'd appreciate the approach I took. Want to see it?
```

## Example 2: Technical Explanation

**Original:**
```
The system uses a filter to smooth out the data because otherwise it looks choppy and bad.
```

**Optimized (professional):**
```
The system applies a smoothing filter to eliminate visual artifacts and improve data continuity.
```

**Optimized (accessible):**
```
We use a filter to smooth the data — without it, the output looks jerky and unprofessional.
```

## Example 3: Persuasive Pitch

**Original:**
```
You should use this because it's better than the others and has more features.
```

**Optimized:**
```
This solution outperforms alternatives by combining [specific advantage] with [unique feature] — delivering [concrete benefit] without [common pain point].
```

## Example 5: Creative Retextualization

**Original:**
```
I'm tired of working on this project. It's taking forever.
```

**Retextualized (frustration → determination):**
```
This project is a marathon, not a sprint. Still here, still pushing.
```

**Retextualized (honest but professional):**
```
I'm hitting the harder stretches of this project. Worth it, but requires patience.
```

## Example 6: Email Tone Adjustment

**Original:**
```
I need this done by tomorrow or we're going to miss the deadline.
```

**Optimized (firm but collaborative):**
```
To stay on track for the deadline, I'll need this by tomorrow. Thanks for prioritizing it.
```

**Optimized (urgent but respectful):**
```
Time-sensitive: we need this tomorrow to hit our deadline. I know it's tight — appreciate the quick turnaround.
```

---

## Key Principles Demonstrated

1. **Clarity over vagueness** - "something" → specific reference
2. **Active voice** - "it will work" → "you're ready to go"
3. **Audience awareness** - Technical vs. accessible versions
4. **Tone matching** - Casual, professional, urgent, collaborative
5. **Purpose alignment** - Each version serves a different goal
6. **Economy of words** - Remove filler without losing meaning
